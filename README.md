# FGO ステータス自動計算機 for atwiki

atwikiのPukiWikiライクモードで使用する、オリジナルFGOサーヴァント用のステータス計算機です。

## 主な機能

- 初期・最大HP/ATKの計算
- レアリティ別の霊基再臨・聖杯転臨値の計算
- 物理型・魔術型ATKへの対応
- ★0～★5への対応（★0の計算条件とレベル構成は★2と同じ）
- SW・SR・DR・N/A・N/Dの計算
- 宝具の「リチャージ量15％以下補正値」の参考表示
- 通常攻撃N/Aの手動補正と適正値注釈
- 通常攻撃用・宝具攻撃用N/Aの分離
- 既存のPukiWikiコードへ計算対象セルだけを自動反映
- パラメーター表のランク文字と橙色ゲージを自動反映
- 結果セル、HP行、ATK行、N/A行、完成コードのコピー
- 入力内容と元コードの同一タブ内保存・ページ再読み込み時の復元

## atwikiへの設置方法（推奨）

1. 計算機を設置する新規ページを作成します。
2. 編集モードを「PukiWikiライクモード」にします。
3. ページの編集権限を「管理者のみ」にします。
4. `atwiki_page_code.txt` の内容をすべてコピーし、ページへ貼り付けます。
5. ページを保存します。

JSは公開GitHubリポジトリからjsDelivr経由で読み込みます。入力内容と元コードは同じタブでページを再読み込みした場合に復元されます。タブを閉じた後や別のタブでは復元されません。

設置コードは次のとおりです。

```text
*FGO ステータス自動計算機

&html(<div id="fgo-status-calculator"></div>)
#include_js(https://cdn.jsdelivr.net/gh/siroihuman/fgo-status-calculator/v1.1.6/FGO_StatusCalculator_atwiki.js)
```

`#include_js`を使用するページは、編集権限を必ず「管理者のみ」にしてください。

`atwiki_inline_page_code.txt` は外部JSを利用できない場合の予備です。atwikiの制限により、ブラウザへの永続保存は行いません。

## 使用方法

1. レアリティ、クラス、成長タイプなどを選択します。
2. コマンドカード枚数、各攻撃のHit数、パラメーターを入力します。
3. 必要な場合だけN/Aの手動補正または通常・宝具N/Aの分離を有効にします。
4. サーヴァント編集ページから、ステータス表を含むコードをコピーします。
5. 計算機の「元のコード」欄へ貼り付けます。
6. 「計算・コード反映」を押します。
7. 「反映後コードを一括コピー」を押します。
8. コピーした完成コードをサーヴァント編集ページへ貼り戻します。

元のコードに含まれる名前、No.、属性、性別、特性などは変更しません。次の部分だけを自動更新します。

- Classアイコン、Rare、Cost、傾向、タイプ
- クラススキル節のクラスアイコン（コメントアウト行を含む）
- レアリティに応じた列設定・レベル見出し
- HP行・ATK行
- 成長タイプ、スター発生率、スター集中度
- Q/A/B/EX/宝具Hit数、DR
- 隠しステータスの宝具種別・色
- 宝具表のCardセル（コメントアウトされたテンプレート行を含む）
- N/A、N/D
- 筋力・耐久・敏捷・魔力・幸運・宝具のパラメーター表

## N/Aについて

「適正N/A」は次の式で計算します。

```text
クラス基本値 × Arts枚数補正 × 魔力補正 ÷ Arts Hit数
```

「15％以下補正値」は次の式で計算し、参考値としてのみ表示します。

```text
15 ÷（宝具カード補正 × 宝具Hit数 × 対象数補正）
```

採用する通常攻撃N/Aを手動変更した場合は、PukiWikiコードへ適正値の脚注を自動追加します。

```text
0.51&footnote(適正値：0.86)
```

通常攻撃と宝具のN/Aを分けた場合は、同じセル内へ次の形式で出力します。

```text
0.55&footnote(通常攻撃時)&br()0.12&footnote(宝具攻撃時)
```

## ローカルで確認する方法

`FGO_StatusCalculator_standalone.html` と `FGO_StatusCalculator_atwiki.js` を同じフォルダーへ置いたまま、HTMLファイルを開くとatwikiへ設置する前に動作確認できます。

## 検算済みデータ

- 赤ずきん／アーチャー／★1／物理型
- ドロシー／ライダー／★1／魔術型
- ガイウス・ユリウス・カエサル・オクタウィアヌス／セイバー／★4／通常・宝具N/A分離
